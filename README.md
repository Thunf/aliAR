# aliAR
> 2016支付宝AR红包助手

> 少抢怡情，多抢伤身，强抢灰飞烟灭，年轻人抢红包要节制😂

## Usage

```node
git clone git@github.com:Thunf/aliAR.git ali
cd ali/

npm i 

grace -s ali
npm run dev

```
